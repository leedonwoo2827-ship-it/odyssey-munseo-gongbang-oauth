# 02 · 실행 (run.bat)

## 실행 — `run.bat` 더블클릭
1. 탐색기에서 `run.bat` 더블클릭.
2. 검은 콘솔 창이 뜨고 서버가 시작됩니다.
3. 잠시 후 기본 브라우저가 자동으로 열립니다. 로그인 화면에서 **[OpenAI (ChatGPT)로 로그인]** 을 누르세요.
   - 처음이거나 계정을 바꾸려면 **🖥️ 로그인 관리 열기** → `codex login` 이 터미널로 떠서 브라우저
     로그인이 진행됩니다. → [OpenAI/Codex 설치·로그인](../openai-codex/install.md)
   - 로그인하면 `http://127.0.0.1:7000/studio` 로 이동합니다. 자동으로 안 열리면 주소창에 직접 입력하세요.

## 종료
- 검은 콘솔 창을 **닫으면** 종료됩니다. (또는 창에서 `Ctrl + C`)

## 화면이 안 보일 때
- "먼저 setup.bat 을 실행하세요" → 아직 설치 전. [01 설치](01_설치_setup.md) 먼저.
- 브라우저에 아무것도 안 뜸 → 콘솔 창이 아직 "Application startup" 중일 수 있음. 5~10초 후 새로고침.
- 포트 충돌(7000 사용 중) → [07 FAQ](07_문제해결_FAQ.md) 의 포트 변경 참고.

## 상단 상태 표시
화면 오른쪽 위의 점이:
- **🟢 OpenAI (ChatGPT) … 연결됨** → 정상 (계정·모델 표시).
- **🔴 연결 안 됨** → codex 미설치/미로그인. 우측 상단 **⚙ 연결 상태** → **[로그인 관리 열기]** → 로그인.
  (점에 마우스를 올리면 원인 표시) → [OpenAI/Codex 안내](../openai-codex/install.md)

## 모델 바꾸기
우측 상단 **⚙ 연결 상태** 모달의 **사용 모델** 드롭다운에서 골라 **[적용]**.
(모델 목록은 `codex debug models` — 예: gpt-5.5 / gpt-5.4 / gpt-5.4-mini)

## 같은 PC에서만? 다른 PC에서도?
- 기본은 **이 PC에서만**(127.0.0.1) 접속됩니다. 안전합니다.
- 사내 다른 PC에서도 접속하려면 바인드 주소를 바꿔야 하며, 이때는 로그인(AUTH) 활성화를 권장합니다.
  → [07 FAQ](07_문제해결_FAQ.md)
