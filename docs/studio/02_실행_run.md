# 02 · 실행 (run.bat)

## 실행 — `run.bat` 더블클릭
1. 탐색기에서 `run.bat` 더블클릭.
2. 검은 콘솔 창이 뜨고 서버가 시작됩니다.
3. 잠시 후 기본 브라우저가 **`http://127.0.0.1:7000/studio`** 로 자동으로 열립니다.
   - 자동으로 안 열리면 브라우저 주소창에 직접 입력하세요.

## 종료
- 검은 콘솔 창을 **닫으면** 종료됩니다. (또는 창에서 `Ctrl + C`)

## 화면이 안 보일 때
- "먼저 setup.bat 을 실행하세요" → 아직 설치 전. [01 설치](01_설치_setup.md) 먼저.
- 브라우저에 아무것도 안 뜸 → 콘솔 창이 아직 "Application startup" 중일 수 있음. 5~10초 후 새로고침.
- 포트 충돌(7000 사용 중) → [07 FAQ](07_문제해결_FAQ.md) 의 포트 변경 참고.

## 상단 상태 표시
화면 오른쪽 위의 점이:
- **🟢 liteLLM 연결됨** → 정상.
- **🔴 LLM 미연결** → 프록시/키 문제. [06 문서](06_liteLLM_연결과_모델.md) 참고. (점에 마우스를 올리면 원인 표시)

## 같은 PC에서만? 다른 PC에서도?
- 기본은 **이 PC에서만**(127.0.0.1) 접속됩니다. 안전합니다.
- 사내 다른 PC에서도 접속하려면 바인드 주소를 바꿔야 하며, 이때는 로그인(AUTH) 활성화를 권장합니다.
  → [07 FAQ](07_문제해결_FAQ.md)
