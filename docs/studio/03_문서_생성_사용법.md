# 03 · 문서 생성 사용법

화면은 좌우 두 칸입니다. **왼쪽 = 유형 선택**, **오른쪽 = 작업/결과**.

## 1단계 · 산출물 유형 고르기 (왼쪽)
- 카테고리별로 버튼이 묶여 있습니다. 위 검색창에 이름을 쳐서 빠르게 찾을 수 있습니다.
- 버튼을 누르면 오른쪽에 그 유형이 요구하는 **입력 슬롯**이 나타납니다.

## 2단계 · 입력 문서 넣기 (가운데)
- 각 슬롯에 파일을 **끌어다 놓거나(드래그&드롭)** 클릭해서 선택합니다.
- `*필수` 표시가 있는 슬롯은 채워야 합니다. (단, 아래 '지시만으로 생성' 참고)
- 지원 형식: **docx · hwpx · xlsx · pptx · pdf**(보조) · txt · md.
  - 파서가 약한 형식은 PDF로 변환해 넣으면 더 안정적입니다.

## 3단계 · 추가 지시 / 메모 (선택 · 채팅처럼)
- "추가 지시" 칸에 자유롭게 적습니다. 예:
  - "10차 일정만 강조해줘"
  - "표는 빼고 서술형으로"
  - "공식적인 보고서 톤으로, 분량은 2페이지"
- **입력 파일이 없어도** 이 지시만으로 생성할 수 있습니다(입력이 필수가 아닌 유형).

## 4단계 · 생성하기
- **생성하기** 버튼을 누르면 잠시 후(보통 10~40초) 아래에 결과가 나타납니다.
- **미리보기**(서식 입힌 본문)와 **다운로드 버튼**(HWPX/DOCX/PPTX/XLSX, 그리고 항상 MD)이 보입니다.
- 노란 **경고**가 보이면 형식 대체 등 안내입니다(예: 한글 미설치 → DOCX로 대체).

## 5단계 · 채팅으로 수정 (다듬기)
- 미리보기 아래 채팅창에 수정 지시를 적고 **보내기**.
  - "3절을 더 자세히", "결론 문단 추가", "수치를 표로 정리"
- 초안이 **다시 만들어지고** 다운로드 파일도 갱신됩니다. 만족할 때까지 반복하세요.

## 결과 파일은 어디에?
- 다운로드 버튼으로 받는 게 기본입니다.
- 원본은 `data/studio/outputs/` 에도 저장됩니다(작업 id 가 파일명에 포함).
