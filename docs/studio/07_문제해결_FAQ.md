# 07 · 문제 해결 FAQ

## 설치/실행

**Q. "Python 3.11 이상이 필요합니다" 라고 나와요.**
A. Python 미설치이거나 PATH 누락입니다. [python.org](https://www.python.org/downloads/) 에서 3.12 설치,
설치 화면에서 **"Add Python to PATH"** 체크 후 `setup.bat` 다시 실행.

**Q. 라이브러리 설치가 실패해요.**
A. 네트워크(사내 프록시) 문제일 수 있습니다. 다시 `setup.bat` 실행. 계속 실패하면 콘솔의
빨간 메시지(패키지명)를 캡처해 공유하세요.

**Q. 7000 포트가 이미 사용 중이에요.**
A. `.env` 의 `APP_PORT` 를 8000 등으로 바꾸고, `run.bat` 의 `--port 7000` 도 같은 값으로 바꾸세요.
브라우저 주소도 그 포트로 접속합니다.

**Q. run.bat 을 눌렀는데 브라우저가 비어 있어요.**
A. 서버 기동 중일 수 있습니다. 5~10초 뒤 새로고침. 콘솔에 `Application startup complete` 가 보이면 준비 완료.

## LLM / 연결

**Q. 우측 상단이 🔴 미연결 이에요.**
A. ① 사내망/VPN 연결 확인 ② `.env` 의 `UBION_LITELLM_KEY` 확인 ③ `venv\Scripts\python sanity-check.py`
실행해 원인 확인. → [06 문서](06_liteLLM_연결과_모델.md)

**Q. 생성 중 오류: "키가 설정되지 않았습니다".**
A. `setup.bat` 을 다시 실행하거나 `.env` 에 `UBION_LITELLM_KEY` 를 채운 뒤 앱 재시작.

## 입력/추출

**Q. 입력 문서에서 "[추출 실패…]" 가 떠요.**
A. 해당 형식 라이브러리 누락 또는 깨진 파일. `setup.bat` 재실행으로 라이브러리 보강.
HWPX/PPTX 가 잘 안 읽히면 **PDF로 변환**해서 넣어보세요(보조 형식).

**Q. 입력이 너무 길어요.**
A. 파일당 약 4만 자까지만 사용하고 이후는 잘립니다(`.env` 의 `STUDIO_MAX_CHARS_PER_INPUT` 로 조정).
핵심 부분만 담은 파일을 넣는 게 품질에 유리합니다.

## 산출물(HWPX 등)

**Q. .hwpx 가 안 나오고 DOCX/MD 로 나와요.**
A. 한컴 한글(또는 pywin32)이 없고, `assets/hwpx_template.hwpx` 양식도 없을 때의 정상 동작입니다.
- 한컴 한글 설치 PC 에서 실행하거나,
- 회사 HWPX 양식을 `assets/hwpx_template.hwpx` 에 넣으면 .hwpx 로 생성됩니다. → [05 문서](05_양식_만들기.md)

**Q. .hwpx 를 한글에서 열었더니 줄 간격/위치가 약간 어긋나요.**
A. 텍스트 길이 변화로 인한 경미한 현상입니다. 한글에서 한 번 열면 자동 재배치되며,
보통 그대로 저장하면 정상화됩니다. 정밀한 서식이 필요하면 한컴 OLE 경로(한글 설치)를 권장합니다.

**Q. 표지 제목이 안 채워져요.**
A. 양식 표지에 `_제목_입력_` 플레이스홀더 텍스트가 있는지 확인하세요.

## 작업/기타

**Q. 서버를 껐다 켜니 이전 결과가 사라졌어요.**
A. 작업 목록은 메모리에 있어 재시작 시 초기화됩니다. 단, 생성된 파일은 `data/studio/outputs/` 에 남아 있습니다.

**Q. 다른 PC에서도 접속하고 싶어요.**
A. `.env` 의 `APP_BIND` 를 `0.0.0.0` 으로, `run.bat` 의 `--host` 도 `0.0.0.0` 으로 바꾸면 사내 LAN 에서
접속됩니다. 이 경우 보안을 위해 `.env` 의 `AUTH_ENABLED=true` 로 바꿔 로그인을 켜는 것을 권장합니다.

**Q. 레시피(버튼)를 추가하고 싶어요.**
A. [04 문서](04_레시피_추가하기.md) 참고 — `.yaml` 한 개 복사·수정이면 끝입니다.

해결되지 않으면 콘솔 창의 빨간 메시지를 캡처해 담당자에게 전달하세요.
