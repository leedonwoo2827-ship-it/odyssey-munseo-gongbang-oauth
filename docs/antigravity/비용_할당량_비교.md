# 모델별 비용 비교 (입력 동일 · 출력은 체급 추산)

이 앱의 용도: **agy(무료/구독)로 여러 모델을 싸게 테스트**해 보고, 가장 좋은 모델을 골라
**그 모델을 API로 연결**해 운영하려는 것. 아래는 그 모델을 **API(종량제)로 썼을 때**의 추정 비용이다.
(agy 구독료는 제외 — 순수 API 단가 기준.)

## 계산 기준
- **입력 토큰: 모든 모델 동일** = 약 **40,000 토큰** (레시피 프롬프트 + 첨부 4개; 모델 바꿔도 입력은 같음).
- **출력 토큰: 모델 체급(관측 분량)으로 추산** (1 A4장 ≈ 약 2,000 토큰 환산).
- 1건당 = `입력40,000 × 입력단가 + 출력 × 출력단가`. 환율 1$≈1,350원.
- 단가·분량은 **근사치**(시점·프롬프트에 따라 변동). (High)/Thinking 은 추론 토큰이 더 들어 더 나올 수 있음.

## 모델별 비교표

| 모델 | 체급/관측 분량 | 출력 토큰(추산) | 입력단가 | 출력단가 | **1건당** | **월 300건** |
|---|---|---|---|---|---|---|
| **Gemini 3.5 Flash** | 가벼움 · ~1장 | 2,000 | $0.3/1M | $1.2/1M | ~$0.014 (**~19원**) | **~5,700원** |
| **Gemini 3.1 Pro** | 중상 · 관측 ~1장 | 2,500 | $2/1M | $8/1M | ~$0.10 (**~135원**) | **~40,500원** |
| **GPT-OSS 120B** | 중(클로드↓) · ~1.5장 | 3,000 | $0.5/1M | $2/1M | ~$0.026 (**~35원**) | **~10,500원** |
| **Claude Sonnet 4.6** | 상 · 관측 ~4장 | 7,000 | $3/1M | $15/1M | ~$0.23 (**~305원**) | **~91,500원** |
| **Claude Opus 4.6** | 최상 · 양↑·표 위주 ~5장 | 9,000 | $15/1M | $75/1M | ~$1.28 (**~1,720원**) | **~516,000원** |

> 입력은 40,000 토큰으로 **모두 동일**, 출력만 체급에 따라 다르게 잡았다. 그래서 비용 차이는
> **출력 분량 + 모델 단가** 에서 갈린다. 예) Opus는 분량도 많고 단가도 높아 1건 ~1,720원.

## 한눈에 — 1건당 / 월(300건)

```
Flash      ~19원   | 월 ~5,700원      (가장 저렴, 간결)
GPT-OSS    ~35원   | 월 ~10,500원     (저렴하나 클로드보다 약함)
Gemini Pro ~135원  | 월 ~40,500원     (균형, 단 분량 적게 나올 때 있음)
Sonnet 4.6 ~305원  | 월 ~91,500원     (분량·품질 좋음)
Opus 4.6   ~1,720원| 월 ~516,000원    (최고 품질, 가장 비쌈)
```

## 해석 / 선정 가이드
- **싸게 대량** → Flash / GPT-OSS (단, 분량·품질은 낮음 → 프롬프트 강화 필요).
- **균형** → Gemini 3.1 Pro (분량 적게 나오면 [팁](팁_분량과_품질_올리기.md)으로 강화).
- **품질·분량 중시** → Claude Sonnet 4.6 (1건 ~305원).
- **최고 품질·표 많은 문서** → Claude Opus 4.6 (1건 ~1,720원, 비쌈).
- 분량이 들쑥날쑥하면 출력 토큰 추산도 흔들리니, **프롬프트를 고정·강화**하면 비교가 정확해진다.

## 참고
- 본 앱(agy OAuth)으로 테스트하는 동안은 **0원**(무료) 또는 구독(월 ₩27,000)으로 모든 모델 사용.
- 위 표는 **"최종 선정 모델을 API로 운영할 때"** 의 예산 추산용. 정확한 단가는 공식 가격표로 확인.
