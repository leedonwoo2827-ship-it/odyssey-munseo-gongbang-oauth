# 모델별 비교 (테스트 → API 연결 선정용)

이 앱의 용도: **agy(무료/구독)로 여러 모델을 싸게 테스트**해 보고, **가장 좋은 모델을 고른 뒤
그 모델을 API로 연결**해 운영(production)하려는 것. 따라서 아래 비용은 **그 모델을 API(종량제)로
썼을 때**의 추정치다. (agy 구독료는 섞지 않음.)

> ⚠️ 품질·분량은 **모델 + 프롬프트 강도**에 크게 좌우된다. 강하게 프롬프팅하지 않으면 같은 모델도
> 들쑥날쑥하다. 그래서 **직접 테스트로 비교**가 답이고, 이 앱이 그 용도다.
> 단가는 시점·공급자에 따라 다른 **근사치** → 공식 가격표로 재확인 권장.

**공통 가정** — 문서 1건: 입력(레시피+첨부 4개) ≈ 3~5만 토큰, 출력 ≈ 3~6천 토큰. 환율 1$≈1,350원.

## 모델별 — 분량/품질 · API 비용 (구독료 제외, 순수 종량제)

| 모델 | 속도 | 품질 | 관측 분량(테스트) | 입력 단가 | 출력 단가 | **1건당 API** | 하루20건→**월 440건 API** |
|---|---|---|---|---|---|---|---|
| Gemini 3.5 Flash | ★★★ | 보통 | (기록) | ~$0.3/1M | ~$1.2/1M | **~25원** | **~11,000원** |
| Gemini 3.1 Pro | ★★ | 좋음 | **약 1장** | ~$2/1M | ~$8/1M | **~160원** | **~70,000원** |
| Claude Sonnet 4.6 | ★★ | 매우 좋음 | **약 4장** | ~$3/1M | ~$15/1M | **~270원** | **~119,000원** |
| Claude Opus 4.6 | ★ | 최고 | **양↑·표 위주** | ~$15/1M | ~$75/1M | **~1,300원** | **~572,000원** |
| GPT-OSS 120B | ★★ | 좋음 | (기록) | ~$0.5/1M | ~$2/1M | **~40원** | **~17,600원** |

> "관측 분량"은 같은 입력·프롬프트로 직접 돌려 나온 페이지 수(테스트 기록용). 예: **Pro ≈ 1장, 소넷 ≈ 4장.**
> 분량이 많다고 무조건 좋은 건 아니고, **요구 분량/품질 대비** 판단. (High)/Thinking 은 추론 토큰이 더 들어
> 위 1건당 비용보다 더 나올 수 있음.

---

## 1건당 비용 산출식
`1건당 = (입력토큰 × 입력단가 + 출력토큰 × 출력단가)`
예) Pro: (40,000×$2 + 5,000×$8)/1,000,000 = $0.08+$0.04 = **$0.12 ≈ 160원**

---

## 모델 선정 가이드 (테스트 관점)
- **분량이 적게 나옴(예: Pro 1장)** → 프롬프트에 "각 절 3~5문단, A4 N장 분량으로" 등 **분량 지시 강화**,
  또는 (High)/Thinking 모델 사용, 또는 **샘플 문서 제공**(분량·문체 맞춤).
- **분량/품질이 좋음(예: 소넷 4장)** → 비용 대비 만족스러우면 그 모델을 **API로 연결**해 운영.
- 최종 선정 후 **production = 해당 모델 API**, 단가는 위 표로 월 예산 추산.

## 테스트 체크리스트
1. 같은 입력·같은 프롬프트로 모델만 바꿔 생성([⚙ 연결 상태]에서 모델 변경).
2. 분량(페이지)·정확도·문체를 위 표 "관측 분량"에 기록.
3. (필요시) 프롬프트 강화/샘플 추가 후 재비교.
4. 만족 모델 확정 → 1건당·월 API 비용 확인 → API 연결.
