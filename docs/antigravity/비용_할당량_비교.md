# 할당량 · 비용 비교 (agy / Antigravity 기반)

이 문서는 **API 키 없이 Google 계정으로 로그인(agy)** 해서 문서를 생성하는 본 앱의
하루 사용 가능량과, 같은 작업을 **종량제 API 키**로 했을 때의 추정 비용을 비교한다.

> ⚠️ **주의(꼭 읽기)**: agy/Antigravity의 **무료/Pro/Ultra 할당량은 Google이 정확한 건수를 공개하지
> 않는다**(고정 건수 → "작업량(work done)" 기반 가변, 5시간마다 리필 + 주간 상한). 아래 건수·금액은
> 공개 정보와 일반적 토큰량으로 한 **추정치**이며, 실제는 계정/모델/문서 길이에 따라 달라진다.
> 종량제 단가는 시점에 따라 바뀌므로 **공식 가격표로 재확인** 권장.

## 2. 1건당 비용 추정 근거 (종량제 환산)

문서 1건 가정:
- **입력**: 레시피 프롬프트 + 첨부 4개(파일당 최대 4만 자 ≈ 합쳐 약 3~5만 토큰)
- **출력**: 계획서 1편 ≈ 한국어 1,500~3,000단어 ≈ 약 3~6천 토큰

| 모델 | 입력 단가(추정) | 출력 단가(추정) | 1건당 추정 |
|---|---|---|---|
| Flash 계열 | ~$0.3 / 1M 토큰 | ~$1.2 / 1M 토큰 | (4만×0.3 + 5천×1.2)/1M ≈ **$0.018 (~25원)** |
| Pro 계열 | ~$2 / 1M 토큰 | ~$8 / 1M 토큰 | (4만×2 + 5천×8)/1M ≈ **$0.12 (~160원)** |

(환율 1$≈1,350원 가정, 단가는 근사치)

## 3. 월간 시나리오 비교 (어느 게 이득인가)

하루 30건(영업일 22일 ≈ **월 660건**) 생성 가정:

| 방식 | 월 비용 | 660건 가능? |
|---|---|---|
| agy 무료 | 0원 | ✗ (하루 ~20건 한도 → 부족) |
| **agy Pro 구독** | **약 27,000원** | ✓ (가변이지만 사실상 커버) ← **가성비 최적** |
| API 키 종량제(Flash) | 660 × 25원 ≈ **16,500원** | ✓ (단, 키 관리 필요) |
| API 키 종량제(Pro) | 660 × 160원 ≈ **105,600원** | ✓ (비쌈) |

**해석**
- **소량(하루 ~20건 이하)** → agy **무료**로 충분(0원).
- **대량(하루 수십 건+)** → agy **Pro 구독(월 ₩27,000)** 이 가장 단순하고 저렴. 키 관리도 불필요.
- 종량제 Flash가 더 쌀 수도 있으나 **API 키 발급·관리**가 필요해 본 앱 취지(키 없음)와 안 맞음.

## 4. 모델 선택 가이드 (앱 [⚙ 연결 상태]에서 변경)

| 모델 | 속도 | 품질·분량 | 할당량 소모 | 추천 용도 |
|---|---|---|---|---|
| Gemini 3.5 Flash | 빠름 | 보통(간결) | 적음 | 초안 빠르게, 대량 |
| Gemini 3.1 Pro (Low) | 보통 | 좋음 | 중간 | 일반 문서 |
| **Gemini 3.1 Pro (High)** | 느림 | 가장 길고 자세 | 많음 | 중요 문서·완성도 |
| Claude Opus/Sonnet 4.6 | 보통~느림 | 매우 좋음 | 많음 | 문장 품질 중시 |

> "(High)"일수록 길고 꼼꼼하지만 **할당량을 더 빨리** 씁니다. 무료 계정이면 Flash 위주, 대량·고품질은 Pro 구독 권장.

## 5. 결론
- 본 앱은 **API 키·과금 0원**(agy OAuth). 비용은 **Google 구독료**가 전부.
- **무료(0원)**: 하루 ~20건까지 연습·소량.
- **Google AI Pro(월 ₩27,000)**: 대량 문서 생성의 **가성비 최적** — 권장.
- 정확한 한도/단가는 비공개·변동이므로, 운영 시 **실제 사용량을 모니터링**하며 조정.
