# 모델별 비용 비교 (입력 동일 · 출력은 체급 추산)

이 앱의 용도: **CLI 로그인(agy=Google / codex=ChatGPT)으로 여러 모델을 싸게 테스트**해 보고,
가장 좋은 모델을 골라 **그 모델을 API로 연결**해 운영하려는 것. 아래는 그 모델을
**API(종량제)로 썼을 때**의 추정 비용이다. (CLI 구독료는 제외 — 순수 API 단가 기준.)

## 계산 기준
- **입력 토큰: 모든 모델 동일** = 약 **40,000 토큰** (레시피 프롬프트 + 첨부 4개; 모델 바꿔도 입력은 같음).
- **출력 토큰: 모델 체급(관측 분량)으로 추산** (1 A4장 ≈ 약 2,000 토큰 환산).
- 1건당 = `입력40,000 × 입력단가 + 출력 × 출력단가`. 환율 1$≈1,350원.
- 단가·분량은 **근사치**(시점·프롬프트에 따라 변동). (High)/Thinking/xhigh 는 추론 토큰이 더 들어 더 나올 수 있음.
- **GPT(OpenAI) 단가는 GPT-5 계열 공개 단가 기준 추정치** — 정확한 값은 OpenAI 공식 가격표로 확인.

> **출력 분량 + 모델 단가** 에서 갈린다. 예) Opus는 분량도 많고 단가도 높아 1건 ~1,720원.

## 해석 / 선정 가이드
- **싸게 대량** → Flash / GPT-5.4-mini / GPT-OSS (단, 분량·품질은 낮음 → 프롬프트 강화 필요).
- **균형** → Gemini 3.1 Pro 또는 **GPT-5.4** (분량 적게 나오면 [팁](팁_분량과_품질_올리기.md)으로 강화).
- **품질·분량 중시** → Claude Sonnet 4.6(1건 ~305원) / **GPT-5.5**(frontier, ~176원).
- **최고 품질·표 많은 문서** → Claude Opus 4.6 (1건 ~1,720원, 비쌈).
- 분량이 들쑥날쑥하면 출력 토큰 추산도 흔들리니, **프롬프트를 고정·강화**하면 비교가 정확해진다.
